# Addictive

Addictive is a modern React + Vite storefront for premium handcrafted chili products.
It delivers a bold, heat-focused shopping experience with animated product cards, spice indicators,
dynamic cart management, and a responsive interface for both desktop and mobile users.

## Project Description

The project is an e-commerce style frontend centered around a signature chili product lineup.
Users can browse products, view detailed product information, add items to cart, remove items,
and review cart totals before checkout.

The visual language emphasizes fire, heat, and intensity using custom animations,
expressive typography, and a tailored dark-themed design.

## Tech Stack

- React 18
- Vite 5
- React Router DOM
- Tailwind CSS
- Shadcn UI components
- Radix UI primitives
- Lucide React icons
- TanStack Query (available for async/server-state patterns)

## Features

- Product listing with animated cards
- Product detail page with spice/heat indicator visuals
- Add to cart from product cards and product detail pages
- "Order Now" and "Shop Now" quick-add behavior
- Cart page with:
- selected products
- per-item price
- quantity tracking
- dynamic total calculation
- remove (delete) item support
- proceed-to-checkout CTA
- Dynamic cart count badge in navbar
- Mobile-friendly navigation with cart icon visible on small devices
- Toast notifications for cart actions

## Project Aim

The aim of Addictive is to provide a high-impact, conversion-friendly shopping UI prototype that:

- showcases a strong brand personality
- demonstrates practical cart state management in React
- offers smooth interactions and responsive behavior
- serves as a reusable base for a real e-commerce product experience

## Run Locally

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```
